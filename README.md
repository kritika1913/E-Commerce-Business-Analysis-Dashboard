# E-commerce Business Analysis Dashboard

## Project Overview
This project analyzes e-commerce sales data using Power BI.

## Tools Used
- Power BI
- Excel

## Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Profit by Region
- Product Performance
- Interactive Slicers

## Key Insights
- Furniture generated highest sales
- East region showed negative profit
- Sales peaked in 2019
